# Fallback Institutional Benchmarks

Use these benchmarks when web search is unavailable. When web search IS available,
prefer searching for current, context-relevant benchmarks tailored to the user's
field, country, and goals — then supplement with these if needed.

## How to Select Benchmarks

Choose 5–8 benchmarks that are meaningful for the user:
1. Match the user's country/region if possible
2. Match the user's field (corporate, academic, aviation, etc.)
3. Include at least one benchmark BELOW and one ABOVE the user's estimated score
4. Create a visual narrative: "you are here relative to these known thresholds"

Suggested markers:
- Australian skilled migration minimum: 6.0 (GT or Academic)
- Canadian Express Entry competitive: 7.0+ (GT, CLB 9+)
- Most top universities: 7.0 (Academic)

**IELTS General Training**. Academic programs require **IELTS Academic**. Include
the appropriate variant based on the user's stated purpose.

## Notes on Currency

These benchmarks were compiled as of early 2026. Institutional requirements change.
When web search is available, always verify current thresholds — especially for:
- Immigration requirements (change with policy)
- University admissions (update annually)
- Corporate mandates (change with leadership)

The corporate benchmarks (Rakuten, Air France, Samsung) and the CEFR-to-TOEIC mappings
are relatively stable and can be used with confidence even without verification.
